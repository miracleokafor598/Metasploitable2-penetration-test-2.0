- # Metaploitable2 Penetration Testing Lab:
- This project demonstrates a penetration testing exercise against the intentionally vulnerable metasploitable2 virtual machine.

## Tools Used:
- Kali Linux
- Nmap
- Metasploit Framework
- virtualBox

## Lab Overview:
- In this lab, the target machine was scanned to discover open services. A vulnerable FTP service (vsftpd 2.3.4) was identified and exploited using Metaploit to gain remote shell access and escalate privileges to root.

## Steps Performed:
- Network reconnaissance with Nmap
- Service enumeration
- Vulnerability identification
- Exploitation using Metaploit
- Shell access obtained
- Privilege escalation to root
